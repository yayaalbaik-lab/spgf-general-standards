# SPGF General Governance Repository

هذا المستودع يحتوي النسخ العامة المعتمدة من قواعد SPGF لاستخدامها كبداية موحدة لأي مشروع برمجي.

## الوثائق العامة

- [SPGF Constitution v6.2](docs/general/SPGF_Constitution_v6.2.pdf)
- [Engineering Standards 23 v1.1](docs/general/Engineering_Standards_23_v1.1.pdf)

## الغرض

هذه الوثائق ليست مخصصة لمشروع واحد. هي إطار عام:

- يحدد مبادئ الحوكمة والانضباط الهندسي.
- يفرق بين مشاريع `Lean` و`Standard` و`Enterprise`.
- يحافظ على المرونة في كمية التوثيق.
- يمنع التسيب في الأمن، سلامة البيانات، القرارات المعمارية، والدين التقني.

## طريقة الاستخدام

عند بداية أي مشروع جديد:

1. استخدم الوثائق العامة كمصدر مرجعي أعلى.
2. حدد بروفايل المشروع: `Lean` أو `Standard` أو `Enterprise`.
3. أنشئ نسخة تطبيقية خاصة بالمشروع بدلاً من تعديل الوثائق العامة مباشرة.
4. ضع تفاصيل المجال والتقنية داخل مستندات المشروع نفسه.

## قاعدة مهمة

الوثائق العامة لا تعدل بسبب احتياج مشروع واحد فقط.  
تعدل فقط عند ظهور درس أو قاعدة تستحق أن تصبح عامة عبر أكثر من مشروع.
